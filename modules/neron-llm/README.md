# neron-llm

Module d’abstraction du moteur LLM.

Responsabilités :
- communication avec Ollama (ou autre moteur LLM)
- gestion des prompts
- application du system prompt

Ce module ne contient aucune logique de décision.
Il est utilisé exclusivement par neron-core.
