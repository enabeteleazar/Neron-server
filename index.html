// architecture

Un seul processus.
Plusieurs agents.

Néron v2 abandonne Docker pour un seul processus Python léger. Chaque fonctionnalité est un agent indépendant.

FastAPI Core — Orchestrateur central exposé sur :8000. Point d'entrée unique.

Agents modulaires — LLM, STT, TTS, Mémoire, Telegram s'activent selon la config.

Ollama natif — Communication directe HTTP avec Ollama local. Aucun intermédiaire.

SQLite embarqué — Base de données locale, zéro dépendance externe.
